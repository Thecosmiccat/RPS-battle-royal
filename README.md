# Rockpaperscissor battle royal

A battle royale rock paper scissors simulator, inspired by Doug Doug

Built with Java script & html
My first time working with java and html

# Features

- Extremely bad
- Excellent time-waster
- Can easily feed your gambling obsession

# Running
Go to the website: https://thecosmiccat.github.io/RPS-battle-royal/

Or open the index.html file in your browser



# How To Use
- chose your bet for which team will win
- after you will get coins if your prediction was right
- you can stop pause using the buttons on the screen
